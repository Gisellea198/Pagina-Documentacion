Introducción a Bulma

Bulma es un marco CSS popular con un sistema simple de cuadrícula flexbox. Se diferencia de otros marcos al adoptar un enfoque más ligero y sin incluir ningún JavaScript, dejando esa decisión totalmente en manos del desarrollador.

El marco Bulma es una solución CSS gratuita basada en el diseño de Flexbox. Con Bulma, la amplia gama de funciones integradas significa un tiempo de respuesta más rápido y menos escritura de código CSS.

Bulma también es completamente de código abierto, lo que significa que el código fuente original de Bulma está disponible gratuitamente para su descarga; no hay límite en lo que usted (y la creciente comunidad de Bulma) puede extender su funcionalidad.

¿Por qué utilizar Bulma?

Estas son algunas de las otras ventajas que creemos que podrían hacer de Bulma CSS su nuevo marco favorito.

  • Bulma es fácil de aprender: Sobre todo, Bulma está diseñado para ayudar a simplificar el proceso de codificación, por lo que tiene sentido que Bulma tenga la facilidad de uso como máxima prioridad.
  • Bulma es modular: Bulma no es un marco de todo o nada; es modular, y eso significa que puedes usar las piezas que quieras y dejar el resto fuera.
  • Bulma es solo CSS: Bulma es un framework exclusivamente CSS; la única salida es un archivo CSS (bulma.css). Y si desea aprovechar la modularidad antes mencionada para personalizar sus variables, puede descargar los archivos fuente individuales .sass. De lo contrario, Bulma funciona bien como una solución plug-and-play.
  • Bulma responde: Puede ser difícil intentar convertir un sitio diseñado para escritorio para usarlo en un dispositivo móvil, pero con Bulma, no es realmente un problema. Bulma es un marco CSS para dispositivos móviles que ofrece una capacidad de respuesta óptima para el sitio.
  • Bulma está bien documentada: El sitio de Bulma proporciona documentación completa para ayudarlo a aprovechar al máximo el marco CSS de Bulma. Los temas incluyen modificadores, diseños, columnas y componentes.
  • Bulma es rica en componentes: Bulma viene con casi todos los componentes que necesita para darle a su sitio una apariencia funcional y nítida, incluidos menús desplegables, barras de navegación, paneles, pestañas y tablas.
  • Bulma es compatible: No todo el mundo utiliza Chrome, por lo que cuando diseñe un sitio, querrá asegurarse de que funcionará bien con la gama completa de navegadores web disponibles.
  • Bulma mejora constantemente: Bulma puede ser relativamente nueva, pero no lo sabrías mirando su historial de actualizaciones. Se están agregando nuevas funciones de manera casi constante y las versiones más nuevas están eliminando rápidamente cualquier error u otros problemas que los usuarios puedan encontrar.

Primeros Pasos

Bulma es una biblioteca CSS . Esto significa que proporciona clases de CSS para ayudarlo a diseñar su código HTML. Para usar Bulma, puede usar el .css archivo precompilado o instalar los .sass archivos para que pueda personalizarlo según sus necesidades.

Opción 1. Use un CDN

Puede importar el archivo CSS directamente desde jsDelivr:

CSS @import

@import "https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma.min.css";

HTML <enlace>

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma.min.css">


La versión RTL también está disponible:

CSS @import

@import "https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma-rtl.min.css";

HTML <enlace>

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma-rtl.min.css">

Elementos

Caja

El box elemento es un contenedor simple con un fondo blanco, algo de relleno y una sombra de caja.

<div class="box">   Estoy en una caja </div>
Estoy en una caja.

Debido a que actúa como un contenedor, puede incluir fácilmente otros componentes, como elementos de formulario:

<form class="box">   <div class="field">    <label class="label">Email</label>    <div class="control">     <input class="input" type="email" placeholder="e.g. alex@example.com">    </div>   </div>   <div class="field">    <label class="label">Password</label>    <div class="control">     <input class="input" type="password" placeholder="********">    </div>   </div>   <button class="button is-primary">Sign in</button> </form>

Componentes Bulma

Barra de navegación

Una barra de navegación horizontal receptiva que puede admitir imágenes, enlaces, botones y menús desplegables.

El navbar componente es una barra de navegación horizontal receptiva y versátil con la siguiente estructura:

  1. navbar el contenedor principal.
  2. navbar-bran del lado izquierdo , siempre visible , que suele contener el logo y opcionalmente algunos enlaces o iconos.
  3. navbar-burgerel icono de hamburguesa , que alterna el menú de la barra de navegación en los dispositivos táctiles.
  4. navbar-menu el lado derecho , oculto en dispositivos táctiles, visible en el escritorio.
  5. navbar-start la parte izquierda del menú, que aparece junto a la marca de la barra de navegación en el escritorio.
  6. navbar-end la parte derecha del menú, que aparece al final de la barra de navegación.
  7. navbar-item cada elemento de la barra de navegación, que puede ser a un div.
  8. navbar-link un enlace como hermano de un menú desplegable, con una flecha.
  9. navbar-dropdown el menú desplegable, que puede incluir elementos de la barra de navegación y divisores.
  10. navbar-divider una línea horizontal para separar los elementos de la barra de navegación.

Barra de navegación transparente

Para integrar perfectamente la barra de navegación en cualquier contexto visual, puede agregar el is-transparent modificador en el navbarcomponente. Esto eliminará cualquier fondo activo o flotante de los elementos de la barra de navegación.

Formularios Bulma

Bulma admite los siguientes elementos de formulario HTML nativo:

<form> <button> <input> <textarea> <label>.

Se admiten las siguientes clases de CSS:

label input textarea select checkbox radio button help

Para mantener el diseño equilibrado de manera uniforme, Bulma proporciona un control contenedor muy útil con el que puede envolver los controles de formulario. Cuando combine varios controles en un formulario , use la field clase como contenedor para mantener el espaciado consistente.

Formulario Horizontal

Si desea un control de formulario horizontal , use el is-horinzontal modificador en el field contenedor, en el que incluye:

  1. field-label para la etiqueta lateral.
  2. field-body para el contenedor input / select / textarea
  3. Puede usar is-grouped o has-addons para los elementos secundarios.
